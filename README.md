# neo

NEO-App will be a Cross-Platform Flutter Mobile Application for Android and iOS devices. This App will be used to control a Controller Device via WiFi. The Controller Device is an external hardware attached to a DSLR Camera with USB Port  App mainly consist of below technical aspects:  Remote Read/Write Settings (WiFi WebSocket API) Live View, Camera Control Element, Gallery File Manager (SMB or other Network File Client Protocol) GPS Geo-tag TensorFlow Lite Model deployment (.TFLite file will be provided), Flickr API (REST based) Bluetooth LE (nRF52 Chip Firmware Update) Device Preset/Settings and User Login Some Custom UI element (Histogram, Curves, Meter-gauge â€¦ etc) Native UI (Material for Android and Cupertino for iOS) Performance optimized
